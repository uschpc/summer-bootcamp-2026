Schedule

Three-day workshop schedule

Day 1

Wednesday, June 24

9:00 AM-4:30 PM

Morning session

  1. 9:00-9:25 AM
    Introduction to CARC Systems

    Presenter: Hao | Duration: 0.5 hr

    An overview of CARC's high-performance computing clusters, including logging in, managing and transferring data, loading and building software, and running and monitoring jobs.

    Notes: overview of services, systems, and Open OnDemand.

  2. 9:30-11:00 AM
    Introduction to the Linux Command Line

    Presenter: Iman | Duration: 1.5 hr

    Basic command-line skills for running commands, navigating file systems, creating and editing files, using pipes and filters, and developing shell scripts.

    Notes: log in via Open OnDemand if needed.

Afternoon session

  1. 1:00-2:15 PM
    Installing and Using Software on CARC Clusters

    Presenter: Cesar | Duration: 1.5 hr

    An introduction to CARC's software stack and Lmod module system, with guidance on managing environments, finding software, loading modules, and building packages.

  2. 2:30-4:30 PM
    Running Jobs on CARC Clusters

    Presenter: Tomek | Duration: 2 hr

    How to run and monitor jobs with Slurm, including resource requests, resource-use evaluation, MPI jobs, job arrays, and job dependencies.

    Notes: based on workshop material; includes command-line job submission, Jupyter notebooks, and Open OnDemand.

Day 2

Thursday, June 25

9:00 AM-4:00 PM

Morning session

  1. 9:00-11:00 AM
    Software Containers with Apptainer

    Presenter: Derek | Duration: 2 hr

    Build and run portable software containers for research computing tasks using Apptainer.

  2. 11:00 AM-12:00 PM
    Managing and Transferring Data

    Presenter: Hao | Duration: 1 hr

    Data management and transfer on CARC systems, including transfer tools, file permissions, file I/O, and backups.

    Notes: covers data movement tradeoffs, security, integrity, and archival practices.

Afternoon session

  1. 1:00-2:30 PM
    Introduction to Git for Version Control

    Presenter: Iman | Duration: 1.5 hr

    A hands-on crash course on Git and GitHub for HPC cluster users, from setup through private repositories, branches, and syncing work between the cluster and GitHub.

  2. 2:30-4:00 PM
    Tools for Many Task Computing

    Presenter: Derek | Duration: 1.5 hr

    Use Slurm job arrays and HyperShell for many-task computing workloads such as file-processing workflows and model parameter sweeps.

Day 3

Friday, June 26

9:00 AM-3:30 PM

Morning session

  1. 9:00-10:30 AM
    R Programming for HPC

    Presenter: Derek | Duration: 1.5 hr

    Improve R code performance through parallel processing, efficient memory use, efficient data processing, profiling, and benchmarking.

  2. 10:30 AM-12:00 PM
    MPI for Python

    Presenter: Cesar | Duration: 1.5 hr

    Basic MPI concepts and commands in mpi4py with coding examples and time for open questions.

Afternoon session

  1. 1:00-2:30 PM
    Building Neural Networks for Deep Learning Applications

    Presenter: Hao | Duration: 1.5 hr

    An advanced introduction to neural networks, including the basics of neural networks and PyTorch code for building deep neural networks.

  2. 2:30-3:30 PM
    Running Deep Learning Applications in HPC Systems

    Presenter: Hao | Duration: 1 hr

    Running deep learning applications on HPC systems, with specific emphasis on Python, conda environments, and related workflows.
